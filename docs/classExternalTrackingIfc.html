Detailed Description

This is an external tracking interface class which provides pure virtual methods, which must be implemented in the derived external tracking program. The derived external tracking class can be registered at OgreControllerBackend to receive data from the GNSS receiver.

See also
OgreControllerBackend

◆ CorrelationValueInterface

enum ExternalTrackingIfc::CorrelationValueInterface

Settings for the output of correlation values CORR_IFC_OFF: Correlation value output is disable CORR_IFC_RCV_ONLY: Correlation value output is enabled. Control values are not expected (no external tracking loops). CORR_IFC_TRK: Correlation value output is enabled. Control values are expected (external tracking loops).
